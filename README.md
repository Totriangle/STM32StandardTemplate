本仓库工程用于快速搭建以STM32为MCU的控制系统工程



本项目主要包括：

| MCU内核版本 | MCU闪存容量 | MCU型号     | 目前是否支持 |
| ----------- | ----------- | ----------- | ------------ |
| Cortex-M3   | 64KB        | STM32F103x8 | 否           |
| Cortex-M3   | 128KB       | STM32F103xB | 否           |
| Cortex-M3   | 256KB       | STM32F103xC | 是           |
